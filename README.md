# Dashboard de Operaciones de Conjuntos

Proyecto desarrollado para el curso **Lógica y Matemática Básica**.

Este dashboard interactivo aplica operaciones de **Teoría de Conjuntos**
sobre una base de datos de estudiantes, utilizando una interfaz web
dinámica y visual.

---

##  Descripción del proyecto

La aplicación permite analizar el acceso de estudiantes a distintos
recursos tecnológicos de una universidad mediante operaciones de conjuntos:

- Laboratorio de Cómputo (A)
- Plataforma Virtual (B)
- Biblioteca Digital (C)
- Club de Programación (D)

Cada operación se representa visualmente y permite filtrar estudiantes
según la lógica matemática correspondiente.

---

##  Operaciones de conjuntos implementadas

- **A ∪ B** → Unión  
- **A ∩ C** → Intersección  
- **B − D** → Diferencia  
- **A △ B** → Diferencia simétrica  
- **Dᶜ** → Complemento

##  Uso del archivo CSV (IMPORTANTE)

El dashboard **no contiene los datos precargados**, sino que trabaja con
un archivo externo en formato **CSV**.

### Para visualizar correctamente la información:
1. Descargue el archivo **`BaseDeDatos.csv`** desde este repositorio.
2. Ingrese al enlace del dashboard.
3. En la interfaz, cargue el archivo CSV cuando el sistema lo solicite.

Una vez cargado el archivo, el dashboard mostrará:
- El total de estudiantes
- Los resultados de cada operación de conjuntos
- La lista de estudiantes filtrados dinámicamente

 **Nota:**  
Los datos contenidos en el archivo CSV son de ejemplo y fueron creados
únicamente con fines académicos.
